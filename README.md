# FocusMode-Youtube-Disabled-shorts-
Realize esta extensión de Google con la intención de no dar la posibilidad de distraerme mientras veo videos de la facultad,o para algún momento en el que necesite prestar mucha atencion al video.  Lo comparto en mi GitHub por si a alguien le interesa,y si le sirve poder utilizarlo. 


Para llevarla a cabo, definí la idea y la lógica del producto, y utilicé una IA como asistente para la implementación técnica, dado que mis conocimientos actuales se centran en HTML básico y programación en C++ (adquiridos en la universidad).

Cómo instalarla? (Modo Desarrollador)
Si querés probar la extensión, seguí estos pasos:
1. Descargá los archivos de este repositorio en una carpeta.
2. Abrí Google Chrome y andá a `chrome://extensions/`.
3. Activá el Modo de desarrollador (arriba a la derecha).
4. Hacé clic en Cargar descomprimida (Load unpacked).
5. Seleccioná la carpeta donde guardaste los archivos.
